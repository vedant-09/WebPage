NodeJs

Developers use Node. js to create server-side web applications, and it is perfect for data-intensive applications since it uses an asynchronous, event-driven model. Now that we know what is Node, let's look at why it is so prevalent in web development

Angular Js

HTML is great for declaring static documents, but it falters when we try to use it for declaring dynamic views in web-applications. AngularJS lets you extend HTML vocabulary for your application. The resulting environment is extraordinarily expressive, readable, and quick to develop.

XAMPP

XAMPP full form is X-operating system, Apache, Mysql, Php, Perl. One of the main features of XAMPP is that it is the simplest and lightest local server. XAMPP is an open-source platform. It works on all major operating systems like Windows, Mac, Linux, etc. Being an open-source platform, you can easily download and install it on your system.
